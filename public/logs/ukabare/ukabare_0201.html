1/1 （火）

A.O!!!

さてまたお相手になってやろうかな。2002 年っつーことで、もう 90's から遠くはなれてしまったぞ。"2001 年" っていうと映画のせいでなじみがあったし、2000 年はミレニアム騒ぎなんかで紛れた感じだったけど、今年からは普通に年越しだったね。もうしばらくなんのイベントもないですね。うひゃー、楽しみだ。なんにもない、ってほんと素敵!!! 馬鹿みたいに毎日同じことのくりかえし。素晴らしい日々が待っている。

とはいえ ベンチに腰掛け、若さを弄びずっと泣いてい られるような年齢でもないので(学生の頃はそんなだったけど）、今年はなんか一発やってやんよ!!!　てか、一発くらいはやりたい!!!　去年はほんと最悪だったもんなぁ…。僕はお疲れになったんで、今日はこれまで。世の中に物申したいこともありません…。

1/2 （水）

楽しみにしていた今年初のお買い物も別になんの収穫もなく終わってしまった…。後ろ向きの消費ばかり。CD と DVD を各 2 枚ずつ。CD は ultra living "trancegresstion" と John Frusciante "TO RECORD ONLY WATER FOR TEN DAYS" で、DVD は "Pink Floyd-THE WALL" と "Sugar Soul Tour 2000-うず-" を買った。しめて 3,800 円。はあ…。金なら持ってんのによ。つーか、どれも特に聴きたい/観たいというものではないねぇ。ジョン・フルシャンテなんて、レッチリ聴いたこともないのに買ってもしゃーないっしょ。てか、「ダーク・サイド・オブ・ロック｣みたいなタイトルの洋泉社から出たムック(探してみたんだけどどこにいったかわからない)で、来日公演中に突然失踪してその後復帰したかと思いきや超ダウナーなをリリースしてみなをびびらせた、ってな感じで紹介されてたから買ってみましたが、まあ、値段分(300 円)の価値しかなかったねぇ…。なんの面白味もなかった。ultra living (なんかよく知らんが音響系+フリージャズみたいな感じの日本人のバンド)はいま聴いてるとこなんでまだなんともいえんね。まあ、悪くはないなあ。コンボ・ピアノさえいればこういうのってもういらないけど、俺としては。HIM とかもいることだし。しかし、7 曲目でオーネット・コールマンの曲をやるみたいなんで、ちょっと期待してますよ、せめて値段分(500 円)くらいには。つーか、すいませんでした!!! ライブ盤が出てから 1 年くらいリリースがなかったんですっかり姉さんのことを忘れておりました!!!Sugar Soul アイコ姉さん!!!もちろん DVD が発売されていたことは当初から知っていましたが、貧乏で買えなかったんです。今回初商い特価(1,500 円)になってたんでやっと買えました。てか、今日売り場で見て DVD の存在を思い出しました…。CD では聴いてたんですが、やはり映像で拝見するとますますお美しく艶やかでいらっしゃる!!! ZEEBRA とかいうオッサンは嫌いだけど、彼とのからみ、大変興奮いたしました!!! やはり CD ではよくわからなかった YOU THE ROCK★ とのコンビネーションというかプチ・コントでも大笑いさせていただきました!!! お召し物も立派で色気があって素敵でしたね。今更ながら嘆息するしかないのが悔しいです!!! 僕は、間違っておりました。松浦亜弥とかいう小娘のことはすっかり忘れて真人間になりました!!! やっぱ、姉さんしかいないっス!!! ライブ、お疲れ様でした。あ、なんか買ってきましょうか？ はい承知しました、いってきます。煙草は｢峰｣でいいんでしたよねっ!!! てな感じで、パシッてきた後、「THE WALL −小金持ちなマザコン・ヒッキーの冒険｣を観た。やぁー、なんていうか、全然期待はずれだったなあ。まあ、そんなに期待してはなかったけど、観るまえにイメージしてたのは「さらば青春の光」のアヴァンギャルド・ヴァージョンってな感じだったから。女性器を模した花と男性器を模していると思われる花とが戦った挙句、マンコ花がチンコ花をぱっくんと喰っちゃうアニメーションはなかなかひどくて笑えたけど。この時代はこんなにそのまんま過ぎる通俗フロイト主義が通用したのだろうか？ てか、どうでもいいや、という感想しかないなあ。10 代の頃に観てたら影響受けたかもしれんが。"THE WALL" すげえ!!! と驚くような年齢はもう過ぎちゃったのね。てか、観ながら考えてたのは、俺もひきこもりたい、ってこと。素敵だなあ、つーか、いいよなあ、ひきこもりって。俺みたいな貧乏人には一生無理だもの。若者のひきこもりが増えたのはそれだけ世の中が豊かになったからだ、彼らは豊かな社会に寄生しているのだ、みたいな意見はよくあるけど、("寄生" っていう言葉はアレだが)実際そうでしょ？それがいい／悪いということではなくて。俺がひきこもって仕事やめたりなんかしたら、半年もたないよ。速攻で餓死。誰も俺を助けられる経済力ないもの。と知ったようなこと書いてるが、小説を 3 冊（村上龍『共生虫』『最後の家族』、阿部和重『ニッポニア・ニッポン』）を読んだという程度のイメージしか持ってはいないんだけど…。でも、その小説の話でいうと、そこに登場するヒッキーたちはほんと経済的条件に恵まれていてうらやましい。でも、話の流れとして、その 3 冊に限らずおそらくほとんどが、引きこもってはいたがこれではいけないと思いいろいろな苦難を乗り越えて社会復帰する、ってなことになるじゃん？ そこで俺はつまづくんだよなあ。え？ なんで？ 金があるならわざわざ仕事なんかしないでおうちでインタネでもしてりゃいいじゃん!!! そうはいかないのは、たいていの家庭は金を持ってるといっても、なにもしない人間を一生養うほど金持ちではないからなんだね。だから、ひきこもりのひともそのうち社会に出なきゃならない、ということになるわけだよね。つまり、社会に出るべきか否かってのはお金のあるなしで決まるわけだよね。でも、お金のためだけにひとは行動するわけではない、というのはいうまでもないことで、他人とのコミュニケーションを切望する等の本能みたいなものもあるんだろうけど。とはいえ、どんな人間もお金がなかったら死ぬよね。それでも、お金を稼ぐ行動をとれないひともいるわけだよね。そういうひとが自殺したりするわけだよね。実際はといえば、ひきこもってはいるが、まだ死んではいないというひとがたいていだと思われるけど、そういうひとの家庭はまだお金が底をついていない(借金の限度も含めて)ということなんだろうね。だから、はやい話、金があるうちはぶらぶら遊んで暮らします、ということなんだよね。うらやましいですな、マジで。まとめると、ひとにとって根源的なことはなにかってーと、生きるか死ぬかでしょ。で、生きるか死ぬかはお金のあるなしに依存するわけなんだけど、それには例外があって、死ぬことは自分で選択できる。ほんでもって、死ぬことを選択していないひとは、いずれ死ぬことを選択するか、金がなくなって死ぬか、使い切れないくらいお金が有り余っているので稼がなくても死なないか、金を稼いで生きるか(別に外に出て働かなくてもいいんだけど)の 4 つしか選択肢はないわけじゃん。まあ、だいたいにおいて経済的な場合分けで理解できる話だと思うんだけど、どうよ？ だから、条件さえ整っているのならひきこもりってすばらしいことじゃない？ 金があって、毎日寝て暮らせる生活を不幸だと思う連中が間違ってるんだよ。ひきこもりたいなぁ…。ちなみに、これだけ明晰に書いたのに「まわりが働けというプレッシャーをかけてくるから安心してひきこもれない｣というひとのために補足しとくと、周囲の連中がうるさいならどこかよそに移り住んでごろごろ暮らせばいいんであって、それができないひとはひきこもるにふさわしくない貧乏人であるわけだから、働くか、金が底をつくまでひきこもった結果餓死するか、自殺するかすればいいんだよ。

1/3 （木）

昨夜のヒッキー論はちょっとおかしいね。金がない→餓死って流れは俺があまりにも貧乏だから自然にそう考えてしまうんであって、まあ、普通はそんなに極端ではないよね。しかし、全体的に正論ではあると思うけど。正論を認識したところでどうしようもないからいろいろ問題が起こるんだけど…。

DOCOMO ショップさんありがとう!!! あなた方が商品の在庫管理をちゃんとやったり商品知識を身につけていたりしなかったおかげで大変助かりました。唐突に Sigmarion がほしくなってお店へ走ったんだけど、質問してもわけわかってないし在庫もないのでお金を使わずにお家に帰った。や、ほんとよかったよ。なんでほしがってたのかいまはもうわからないもの。買っても使わないから、どうせ。外出先で PC 使う用事なんてないもんね。お金が浮いたんで、帰りに "Pertition Magic" (11,800 円)を買った。こないだ買った VAIO に Linux を突っ込もうかと思って。はたしてうまくいくのだろうか。

もうお疲れだよほんと。明日から仕事だしよ。はやく 2002 年終わんねぇかな。

1/4 （金）

マジ目が死ぬ!!! 新年早々トップ・スピードで仕事始め。朝から晩まで休みなし。ついでに就業後は飲みに出て。もう死んじゃう、ってんでどこかよくわかんないとこで煙草吸いつつ寝てたらみんないなくなってた。目の奥がずぅずぅするんでモニターの明かりがつらすぎる。てか、昨夜日記を UP してから Linux のインストールに取り掛かったのだが、これがまたなんちゅうか、うまくいってるのかどうかもわからん、っつーか、GUI 環境が立ち上がってくれないんだよ。というわけで 2 時間くらいしか眠れなかったのね。いまや、もう文章をちゃんと整理しながら書く気力もない(もとよりほぼないのに)のでなにを書いてるのかまるでわかんないが、そういったわけで今日は帰りしな TSUTAYA に寄って Linux 本とリーナス・トーバルスの自伝みたいな本を買いましたよ。だが、目が死んでるから一文字たりとも読めないんだよ!!!

>追記：昨夜はほんとに疲れ果てていたようで、↑まで気合で打ち込んだものの、更新する体力が残っていなかったようだ。PC も起動されたまま放置されていたし…。よって、現在 5 日の朝、とりあえず更新しておこう、昨日の俺の頑張りに免じて。

1/5 （土）

今日も Linux さんと闘争。やっと GUI 環境を達成。てか、そんなことを 達成 といわねばならないというのは、俺があまりにもなんも知らんからってことはおいといて、なんか楽しいね。なぜ PC がこんなにも普及したか、との問いに対してどこかの学者は、PC は人間の「障害を克服することによる達成感を求める本能」を適度に刺激する程度にトラブルが多いからだ、と答えたという。一理ある感じがするね。わけわかんないことになったら、それがほんとにどうしようないことならあきらめるけど、解決できそうなことならアドレナリンが噴出するもの。それでまた金をばかばか使って資料やらソフトやら買いこんでるもの。

正月の安売りの余波で「太陽を盗んだ男」の DVD が 2,000 円で叩き売られていた!!! もちろん即購入。やべぇよ、マジで。とはいえ時間がなくてまだ観てないんだけど。それに、「プロコフィエフ＆フランク：フルート協奏曲　ゴールウェイ＆アルゲリッチ：演奏」を買った。こちらも時間がなくてまだ聴いてないが。ほんと時間が足りなさ過ぎる!!! 仕事なんかしてるからだ!!! なんのために金を稼いでいるというのか!!!

で、学生の頃の先輩から年賀状が届いたんだけど、なんか久々にその頃のことを思い出すと、なんかなぁ、という感じになりましたよ。よくよく思い返さずとも、俺ってば全然変わってないじゃんすか!!! 卒業してから数年、一応なんとか職を得て毎日強制的に働かされているというのに。髪の生え際も後退しているというのに。今日はいとこの成人式のお祝いがあって、そうでなくても街中を歩けば新成人どもがけばけばしいヘア・スタイルで得意気に闊歩していてかなり鬱。久しぶりに昔の友達と会うからってんでみんな気合入れるわけよ、田舎者の習性として。つーか、僕は精神年齢が低いせいで、彼/彼女らのことを年下で低能で下等な存在であるとは思えないんだけど。

それでまた買い物でもせにゃおさまらねーよ、という気になって CD 屋に行ってヴェルヴェット・アンダーグラウンドのライブ盤を購入。いやぁ、ほんとかっこいいね!!! 昔、学生の頃にファーストを聴いたときはあんまりよさがわかんなくて、いいなと思うようになったのは 2 年位前からなんだが…、なんてことを考えてまたダウナーな状況。あと、元ちとせっていう歌手の CD を買っていま聴いてるけど、なかなかよろしいですな。まだ若いなあという感じの荒削りなスキルだけど、これから経験積めばかなりすごくなるんじゃね？という気がする。しかし、僕にはアイコ姉さんがいるので、君に気を回す余裕がないんだ、ゴメン!!!

今日もほんとにお疲れ。まずは Linux からインタネに接続できるようにしよう!!! まだできないのよ、これが。なぜ？ LAN カードはちゃんと読み込まれているし設定もしたし、PPPoE の設定もちゃんとやったはずなのに…。このサイトを Linux 上で更新することをとりあえずの目標にまた闘争の日々。

1/6 （日）

とりあえず、Linux さんは CD を聴くことくらいはお許しくださったようです。てか、ミキサーの CD の音量が 0 になってた。なんなんだよ!!! しかしこれで安心してはならない。いまだに USB のマウスが使えないものだからタッチパッドで操作せにゃならんし、もちろんインタネには接続できないのだ。世の中には大変親切な方がいらっしゃってもろもろの問題の解決法をご教示くださるのだが、その通りにやっているつもりなのにまだなにも解決していないのだ。つーか、さっき IE が壊れた。「ブラウザクラッシャー研究所」ってとこ（なんか怖いのでリンク張れない）で、楽しいパズルゲームに興じてたら僕はまだまだ初心者なので大変なことになってしまいました。面白かったけど。

1/8 （火）

グギャ〜ヴォォ!!! てな感じで、グラインド・コアな叫びではいってみたが、如何？ コピー・バンドのための練習。
