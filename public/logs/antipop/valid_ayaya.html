Mission

いまや誰もがあややを心から愛している。Web を少し経巡ってみれば、あたり一帯を彼女を賛美する文書が充たしていることは、たとえアンチであっても認めざるを得ないだろう。しかし、その文書は Web 上に置かれるにふさわしい記法でもって作成されたものだろうか？ invalid な HTML 文書または XHTML 文書で以て讃えられることを果たしてあややが喜ぶのか。ともすれば、冒涜とすらいえるのではないか。それは、彼女を愛しているといいながら電波な脅迫状を毎日送り続けることと等しい、神をも恐れぬ所業なのだ。真にあややを愛する我々は、愛を告白するに valid な文書を以てせねばならない。

A3C HTML Validation Service

…というわけで、Valid Ayaya な文書を作成するべく、A3C HTML Validation Service を利用して皆さんの Web site を validate しましょう。見事 valid であると判定されたサイトの管理人さんは （©猪川まことさん@豚の戰爭 ）こんなステキなバナーを貼ることができます。このバナーを貼っているサイトをみたあややは 从‘ o‘从＜ドッキドキ！ LOVE メールしちゃうぞ！ と大喜び間違いなし。あややは valid なサイトが大好き！

ていうか「おまえのサイトがいちばん valid じゃないではないか」とかいって虐めるのはなしの方向で。
